High Speed Racers is a game intended to showcase the essentials of what sailing is about.
It has 4 tutorial stages that helps the user to familiarize himself with the game controls and the different 
directions that are possible to sail in.
High Speed Racers is written in Python. Opening and running the code in the file "Main.py" will 
let the user run the game.

THe 3rd party library Pygame was used. It can be installed by following the directions on this page:
http://www.pygame.org/download.shtml. Please follow the directions on the site!

Thanks and hope you enjoy the game!
